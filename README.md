# Smart Adaptive Environment Monitoring & Decision System

An intelligent IoT and AI-driven framework designed to monitor environmental conditions, learn complex patterns, and execute autonomous decisions based on real-time sensor data, as outlined in **Scenario 1.pdf**.

---

## 🚀 Key Features
* **Multi-Sensor Monitoring**: Tracks temperature, gas levels, light, and humidity[cite: 1].
* **Intelligent Classification**: Categorizes environments into Normal, Warning, or Dangerous states using SOM[cite: 1].
* **Autonomous Response**: Executes adaptive decisions like activating fans, alarms, or irrigation[cite: 1].
* **Graph-Based Modeling**: Utilizes **Graph Neural Networks (GNN)** to model sensor relationships[cite: 1].

---

## 🧠 AI & Machine Learning Stack

The system employs a sophisticated pipeline to process data and optimize performance[cite: 1]:

| Component | Methodology | Purpose |
| :--- | :--- | :--- |
| **Preprocessing** | PCA | Dimensionality reduction and noise removal[cite: 1]. |
| **Clustering** | SOM | Environment state classification[cite: 1]. |
| **Adaptive Learning** | ART1 / ART2 | Detection of new, unseen binary or continuous patterns[cite: 1]. |
| **Prediction** | RBF Network | Forecasting temperature trends and gas danger levels[cite: 1]. |
| **Optimization** | Genetic Algorithm | Fine-tuning ANN weights, RBF centers, and Fuzzy rules[cite: 1]. |

---

## 🛠 How it Works

1. **Data Acquisition**: Sensors act as nodes in a GNN to capture spatial relationships[cite: 1].
2. **Pattern Recognition**: The system uses Fuzzy Logic to convert sensor data into human-like rules[cite: 1].
3. **Continuous Evolution**: Through **Reinforcement Learning**, the system improves its decision-making accuracy over time[cite: 1].
